# Picture Description Test Images

This directory contains images for the Picture Description section of the English Placement Test.

## Required Images

Place the following image files in this directory:

### B2 Level Images

- **pd-01.jpg** - Industrial/workplace scene
  - Description: "A man is operating industrial machinery while wearing protective gear."
  - Requirements: Real photograph, professional setting, clear subject

- **pd-02.jpg** - Business presentation scene
  - Description: "A woman is presenting data to a group of seated colleagues."
  - Requirements: Office/meeting room setting, multiple people visible

### C1 Level Images

- **pd-03.jpg** - Laboratory/research scene
  - Description: "A researcher is recording observations while handling lab equipment."
  - Requirements: Scientific setting, professional context

- **pd-04.jpg** - Transportation/commuter scene
  - Description: "Commuters are boarding a train on a crowded urban platform."
  - Requirements: Public transportation setting, multiple people, urban environment

### C2 Level Images

- **pd-05.jpg** - Professional kitchen scene
  - Description: "Kitchen staff are collaborating on a complex dish during a busy service."
  - Requirements: Commercial kitchen, multiple staff members, active work scene

## Image Specifications

- **Format**: JPG
- **Dimensions**: Minimum 800x600px, recommended 1200x800px
- **Aspect Ratio**: 3:2 or 4:3 preferred
- **Style**: Real photographs (not illustrations)
- **Color**: Black and white or desaturated preferred to reduce visual bias
- **Quality**: High resolution, clear subjects, good lighting
- **Content**: Professional or social scenes showing real-world activities

## Image Guidelines

1. **Clarity**: Subjects should be clearly visible and identifiable
2. **Complexity**: Images should show enough detail to support multiple plausible descriptions
3. **Neutrality**: Avoid culturally specific or potentially biased imagery
4. **Professionalism**: Focus on workplace, educational, or public settings
5. **Action**: Show people engaged in activities, not static poses

## Sources

You can obtain suitable images from:
- Stock photo websites (Unsplash, Pexels, Pixabay - free commercial use)
- Professional photography collections
- Educational resource libraries
- Ensure you have proper licensing for any images used

## Testing

After adding images, verify:
- Images load correctly in the test interface
- File sizes are optimized (< 500KB per image recommended)
- Images display properly on both desktop and mobile layouts
- Descriptions accurately match the image content
